В начале программы импортируются все необходимые библиотеки и модули.
После идут классы, основная часть программы.

Класс VoiceHelp основной класс в которой лежат все необходимые функции для 
выполнения работы программы.

При инициализации класса определяется место проживания, язык, API ключ благодаря
которому будет находится погода, нахождение температуры, состояния ветра, облаков
данного дня, времени и т.д

В классе существуют следующие вспомогательные функции:
1) task_reform С помощью данной фунции программа будет правильно преображать запрос
  для пользователя и функций выполнения команд
2) vh_help Функция которая будет заполнять все файлы необходимые для программы.
   Своеобразный "помощник голосового помощника" (возможна разбивка этой функции
   на несколько в зависимости от необходимости)
3) start_ui Функция запука интерфейса программы

И функции выполняющие команды пользователя:
1) system Выполняет функции связанные только с системой
2) browser Выпоняет функции отвечающие только с запросами в интернет
3) command Выпоняет функции которые нельзя отнести в какую либо категорию
   но могут быть нужны для пользователя
